Mengapa Helpdesk ULBI ?

Cepat

Permasalahan akan direspon sesuai standar perjanjian tingkat layanan (SLA) maksimal 1 hari kerja.

Terintegrasi

Akun telah terintegrasi dengan Pamong Desa sehingga mempermudah pengguna dalam mengakses sistem.

Responsif

Dukungan sistem WhatsApp, pengguna dapat memonitor progess permasalahan secara real-time

Helpdesk ULBI

Helpdesk ULBI merupakan sistem layanan bantuan permasalahan terkait sistem aplikasi yang ada di ULBI yang ditujukan bagi Mahasiswa ataupun Dosen. Pengguna dapat mengajukan permasalahan setelah memiliki akun valid.

Layanan Kami

Layanan Sistem Informatika

Menyediaka bantuan teknis mengenai iTeung dan dukungan layanan aplikasi TIK.

Layanan Akademik

Menyediakan bantuan layanan terkait akademik dengan fokus pada pelayanan terbaik untuk membantu memenuhi kebutuhan akademik anda.

Layanan Keuangan

Menyediakan bantuan layanan khususnya dalam hal terkait pembayaran perkuliahan dan layanan keuangan lainnya untuk mahasiswa.

Layanan Infrastruktur

Menyediakan layanan terkait jaringan internet dan mengenai perangkat lunak.

Layanan Program Studi

Menyediakan layanan terkait prodi yang dirancang khusus untuk mendukung pengembangan program studi.

Cara Kerja

1. Official Nomor Helpdesk
Pastikan nomor yang dikontak adalah official dari kami.

2. Open Chat
Pengguna Melakukan chat WhatsApp ke nomor wa helpdesk, maka akan otomatis muncul menu utama
3. Pemilihan Menu Utama
Pilih menu yang sesuai dengan kebutuhan anda, untuk kembali ke menu utama tekan 0
4. Soal sering ditanya
Pengguna Mendapatkan Informasi tentang Soal Sering Ditanya di menu tersebut
5. Bantuan Admin
Pengguna Dapat Menghubungi langsung admin pamong desa Jika Ada Pertanyaan Lanjutan di menu kontak admin
6. Close Tiket dan Feedback
Tiket Ditutup oleh Admin atau user dengan menekan angka 0, kemudian user memberikan feedback
